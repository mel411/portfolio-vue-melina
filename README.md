# Portfolio Vue.js – Melina Yorgova

## Présentation

Ce projet est un portfolio développé avec Vue.js dans le cadre de ma formation de développeuse web full-stack.

L’objectif était de concevoir un site moderne, responsive et structuré permettant de présenter mon profil, une sélection de projets réalisés pendant ma formation, ainsi qu’un formulaire de contact.

Le portfolio met en avant plusieurs travaux réalisés avec différentes technologies, notamment React, AngularJS, Node.js, Symfony et Figma.

---

## Fonctionnalités

- Page d’accueil avec présentation
- Section projets avec cartes interactives
- Fenêtre modale pour afficher le détail de chaque projet
- Section contact avec formulaire
- Navigation fluide entre les sections
- Design responsive
- Header et footer réutilisables
- Page 404

---

## Technologies utilisées

- Vue.js

## Projets présentés

## Prérequis

Avant de lancer le projet, il faut avoir installé :

- Node.js
- npm

---

## Installation

Cloner le dépôt :

```bash
git clone https://github.com/mel411/portfolio-vue-melina.git

Entrer dans le dossier du projet :

cd portfolio-vue-melina

Installer les dépendances :

npm install

---

**## Lancement du projet**

Pour démarrer le serveur de développement :

npm run dev

Puis ouvrir dans le navigateur l’adresse indiquée dans le terminal, généralement :

http://localhost:5173
```
